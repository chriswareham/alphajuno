Roland MIDI System Exclusive Control

Control That Juno!

This guide will explain how you can use MIDI System Exclusive Messages (SysEx) to achieve Real-Time control of all editable parameters of the Roland Juno 106, Alpha Juno 1, 2 and MKS-50. The MIDI SysEx information can be recorded, transmitted and replayed just like MIDI controller data, giving you totally automated and controllable filtering and editing and parameter editing for these classic synths!

All you need is a Hardware or Software Sequencer that is capable of outputting SysEx MIDI messages. Sequencers like Performer, Cubase, Logic, and hardware like the Roland PG-300 Controller have this ability. In the following guide, you will get the info you need to set-up Continuous Controllers or Mixer Maps that you can use within your Sequencing program to provide, record and playback real-time editing.

Roland Juno 106 Control

The Juno 106 will transmit SysEx messages through its MIDI Out port when the MIDI Function switch is set to position 3 (III). Any slider moves or buttons pushed on the Juno 106's front panel will be transmitted and can be recorded into a sequencer - and played back! Any on-board editing you transmit and record into your sequencer will always be re-applied to the Juno 106 during playback.

However you may want to create your own programmer/controller for your Juno 106 using your Sequencer program. Simply set-up a Continuous Controller that will send SysEx MIDI Messages to your Juno 106 and use the following code-chart to create controllers for the Juno-106 parameters you want to edit.

I. First you must set the Juno 106's Midi Mode switch to III (on the rear).

II. THESE ARE THE COMPONENTS OF THE SYSTEM EXCLUSIVE MESSAGE:

F0 [Exclusive]
41 [Roland ID#]
32 [Function Type]
0N [N+1=MIDI channel (N=0-15, Chan 1=00)]
XX [Parameter number (0-09, 0A-0F)] see below
YY [Variable value (0-127)] Should be left at 00
F7 [End of Exclusive]

III. HERE'S AN EXAMPLE:

A SysEx message for editing the VCF Resonance Parameter:

F0 41 32 00 06 00 F7

"06" Is the SysEx code for the VCF Resonance parameter

Roland Alpha Juno 1,2 & MKS-50 Control

The Alpha Juno 1, 2 and MKS-50 transmit system exclusive midi messages through the Midi Out port whenever you do any editing via the alpha-dial or press any buttons. These SysEx messages can be recorded into a sequencer and played back! And any editing you've recorded will always be re-applied during playback and can be further fine-tuned.

You may be able to create your own external programmer/controllers for your Juno using your Sequencer or computer. Simply create a Controller that will send System Exclusive MIDI Messages to your Juno and use the following codes to create controllers for the parameters of your choice!

I. First scroll through the Midi menu and turn Exclusive Messages ON.

II. THESE ARE THE COMPONENTS OF THE SYSTEM EXCLUSIVE MESSAGE:

F0 [Exclusive]
41 [Roland ID#]
36 [Individual Tone Parameters]
0N [N=MIDI channel (N=0-F, Chan 1=00 Chan 16=0F)]
23 [Format type]
20 [unknown]
01 [unknown]
XX [Parameter number (0-23)] see below
YY [Value (0-127)] Can be left at 00
F7 [End of Exclusive]

III. HERE'S AN EXAMPLE:

A SysEx message for editing the VCF Resonance Parameter:

F0 41 36 00 23 20 01 11 00 F7

"11" Is the SysEx code for the VCF Resonance edit parameter

"00" Is the SysEx code for the variable value of the current edit parameter
